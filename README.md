# 🚀 TID-INVEST v2

**Plateforme d'investissement fintech moderne avec système VIP, parrainage, recharge, retrait et panneau administrateur sécurisé.**

---

## 📋 Stack Technique

- **Backend** : Node.js + Express
- **Database** : PostgreSQL + Prisma ORM
- **Frontend** : React + Vite (Phase 9)
- **Authentification** : JWT + bcrypt
- **Automation** : node-cron
- **Sécurité** : Helmet, CORS, bcryptjs
